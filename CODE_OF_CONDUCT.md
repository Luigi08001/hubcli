# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

Be respectful, assume good intent, keep discussion on the technical merits. Harassment, demeaning comments, and gatekeeping are not welcome.

## Scope

Applies to all project spaces — repo issues, PRs, discussions, and any channel where someone is representing the project (talks, conference booths, social media referencing the repo).

## Reporting

If you experience or witness behavior that violates this code, open a confidential report by emailing the maintainers or by using GitHub's private reporting feature on the repo.

Reports are read, acknowledged, and acted on. Retaliation against a reporter is itself a violation.

## Enforcement

Maintainers may, in order of escalation:

1. Edit or remove the offending content with a short note on why.
2. Privately warn the contributor and ask for a change in behavior.
3. Temporarily block the contributor from interacting with the project.
4. Permanently ban the contributor from the project.

Severity of the response scales with the severity of the incident. First-offense good-faith mistakes are handled with a conversation, not a ban.
